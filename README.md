### `original_source/`
Contains the original source file(s) selected for mutation testing.  
These files are copied from the Home Assistant repository.

### `mutants/`
Contains a single file with **3-line unified diffs** for each mutant created.  
Each mutant is produced by applying **one mutation operator** at **one location** in the source file.

### `reports/`
Contains the mutation testing report including:
- Mutation operators used
- Test results (killed vs survived mutants)
- Analysis of surviving mutants
- Effectiveness calculations
